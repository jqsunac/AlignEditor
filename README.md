<h2>Usage</h2>
<p>Download all files into the directory can use CGI (.pl extension).</p>
<pre>
ming/
  |--- index.pl
  |--- gene.pl
  |--- mapping.pl
  |--- setting.pl
  |--- usage.pl
  |--- CONFIG
  |--- lib/
  |--- GENE/
  |--- script/
  |--- style/
  |--- template/
</pre>

<p>Then edit CONFIG file.</p>
